# **App Name**: AI TextForge

## Core Features:

- Text Input Interface: Provide a clean, multi-line input area for users to type their prompts for the AI model.
- AI Text Generation: Utilize a server-side API endpoint powered by Google's Gemini model to process user prompts and generate contextually relevant text. This feature employs the AI as a tool for creative reasoning and content creation.
- Generated Text Display: Display the AI-generated text in an easily readable format, ensuring good readability for potentially lengthy outputs.
- Dynamic Loading Indicator: Show a clear visual loading state to indicate when the AI is processing the request and generating text, providing immediate feedback to the user.
- Robust Error & Feedback Messaging: Implement user-friendly error messages for cases such as empty prompts, API call failures, or rate limits, along with helpful user guidance.

## Style Guidelines:

- A dark theme using a primary deep purple (#A667E4) to convey sophistication and creativity. The background color is a subtle, dark charcoal-purple (#26212C), ensuring high contrast for readability. An analogous vibrant blue (#5252E0) serves as an accent for interactive elements.
- Headlines and prominent text will use 'Space Grotesk', a sans-serif font, for a modern, tech-inspired feel. Body text and longer content will use 'Inter', a neutral grotesque-style sans-serif, ensuring clear and objective readability.
- Use a set of minimal and modern line-based icons for key actions (e.g., generate, copy, clear) to maintain a clean aesthetic and improve user experience.
- A responsive, single-column layout optimized for text-centric content, with clear separation between input, actions, and output areas, adaptable across various screen sizes.
- Subtle and quick animations, such as fade-ins for generated content and discreet spinner for loading states, to enhance user feedback without being distracting.
